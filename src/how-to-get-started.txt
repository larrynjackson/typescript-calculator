step 1: install nodejs, google download and install nodejs
	run npm -v to see the version installed
step 2: install typescript - npm install -g typescript
	tsc -v to see the version
step 3: get an editor like visual studio code
step 4: in the terminal window of visual studio code create a file
	called main.ts -
		export {}
		let message = "hello world";
		console.log(messsage);
step 5: compile main.ts - tsc main.ts creates file main.js
step 6: run main.js - node main.js
